# 📋 CityMotion Frontend - Lista de Tarefas

---

## ✅ Concluído

- [x] **Integração com Backend via NexusBridge:** Todas as chamadas de dados passam pela ponte.
- [x] **Autenticação JWT:** Login, logout e refresh de token implementados.
- [x] **RBAC no Frontend:** Sidebar e páginas filtradas por perfil de usuário.
- [x] **Painel de Infraestrutura:** Configuração de banco, proxy, SMTP e servidor.
- [x] **Terminal TTY:** Console interativo com comandos de manutenção.
- [x] **Crachá Virtual:** QR Code dinâmico com validação online.
- [x] **LGPD:** Banner de consentimento e central de privacidade.
- [x] **PWA:** Manifesto e metatags para instalação como app.
- [x] **Dashboard Adaptativo:** Painéis diferentes para admin, gestor, motorista e mecânico.
